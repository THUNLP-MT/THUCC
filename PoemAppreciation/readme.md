## 运行环境

本软件在如下环境经过测试，但应能在兼容的环境下运行：

-  64位Linux 
-  Python 3.6 

## 使用说明
- 下载文件后，进入`src`目录
- 执行`python PoemAppreciationAPI.py YOUR_POEM`
- 其中 `YOUR_POEM` 是待赏析的古诗
